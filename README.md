Explorando a Sintaxe: Primeiras Práticas com Java

Exercícios de lógica de programação focados em cálculos matemáticos e entrada/saída de dados, desenvolvidos como a primeira prática utilizando a linguagem Java.

Este projeto representa o primeiro contato prático com a linguagem, consolidando conceitos fundamentais para a construção de algoritmos simples e funcionais.

Aprendizados

Durante o desenvolvimento dessas atividades, foram aplicados conceitos essenciais da base do Java:

Saída de dados (System.out.println - "sout")
Utilizado para exibir mensagens e resultados no console, sendo fundamental para visualização das informações processadas.
Entrada de dados (Scanner)
Permite capturar valores digitados pelo usuário, tornando os programas interativos e dinâmicos.
Estruturas de repetição (loops)
Utilização de estruturas como for e while para repetição de processos, validação de dados e controle de fluxo.

Esses conceitos são a base para evoluir em lógica de programação e desenvolvimento em Java.

Exercícios

Utilize o Visual Studio Code ou similar para executar os algoritmos.

Leitura de Número
Faça um programa que peça um número e exiba a mensagem:
O número informado foi [número].
Operações Matemáticas
Faça um programa que peça dois números e imprima:
Soma
Subtração
Multiplicação do primeiro pelo segundo
Média de Notas
Faça um programa que solicite 4 notas bimestrais e calcule a média.
Cálculo de Combustível
Faça um programa que peça o preço do litro da gasolina e a quantidade de litros vendidos, exibindo o valor total a ser pago.
Cálculo de Viagem (Eurotrip)
Desenvolva um programa que:
Solicite o preço da viagem para três destinos: Alemanha, Portugal e Itália
Solicite a quantidade de pessoas
Calcule e exiba o valor total da viagem considerando todos os destinos
